# Champions League Drawer

Show drawing results of UEFA Champions League round of 16

## Procedure

The round of 16 pairings are determined by means of a draw. The round of 16 is played under the knockout system, on a home-and-away basis (two legs). The UEFA administration ensures that the following principles are respected.

  - Clubs from the same association cannot be drawn against each other.
  - The winners and runners-up of the same group cannot be drawn against each other.
  - The group-winners cannot be drawn against each other.
  - The runners-up cannot be drawn against each other.
  - The runners-up must play the first leg at home.

[Source](https://www.footballseeding.com/details/champions-league-format/)

This procedure also used on quarter final

## Installation

Require [Node.js](https://nodejs.org/) to run
Clone this repo and install the dependencies

```sh
$ cd champions-league-drawer
$ npm install 
$ npm run start
```

if you are using [Yarn](https://yarnpkg.com/en/)

```sh
$ cd champions-league-drawer
$ yarn install 
$ yarn start
```

open your terminal and you will see the drawing table
